## 📱 Sobre el Proyecto

**Flow-chat** es una plataforma de mensajería de última generación desarrollada como parte del curso de Udemy. Esta aplicación aprovecha el poder de **Flutter** para una interfaz de usuario multiplataforma hermosa y **MongoDB** para un almacenamiento de datos flexible y escalable.

### 🎯 Objetivo
Crear una experiencia de comunicación fluida y en tiempo real donde los usuarios puedan conectarse instantáneamente, gestionada por un backend robusto y verificada mediante pruebas rigurosas.

---

## 🚀 Características Principales

- **⚡ Comunicación en Tiempo Real**: Mensajería instantánea potenciada por web sockets.
- **🎨 UI/UX Premium**: Diseño moderno con "glassmorphism", transiciones suaves y un diseño responsivo.
- **🔐 Autenticación Segura**: Sistemas robustos de inicio de sesión y registro de usuarios.
- **📦 Base de Datos Escalable**: Gestión flexible de datos con MongoDB.
- **🧪 Fiabilidad de la API**: Suite de pruebas completa utilizando Postman.

---

## 🏁 Comenzando

### Requisitos Previos

*   Flutter SDK (^3.0.0)
*   Dart SDK
*   Instancia de MongoDB
*   Cliente de Postman

### Instalación

1.  **Clonar el Repositorio**
    ```bash
    git clone https://github.com/tu-usuario/Flow-chat.git
    cd Flow-chat
    ```

2.  **Instalar Dependencias**
    ```bash
    flutter pub get
    ```

3.  **Ejecutar la App**
    ```bash
    flutter run
    ```

---

## 🧪 Pruebas con Postman

Para asegurar la fiabilidad de nuestros servicios de backend, utilizamos **Postman** para las pruebas de la API.

1.  Abre Postman e importa la colección que se encuentra en la carpeta `test/api_collection`.
2.  Configura tus variables de entorno (Base URL, Secret Keys).
3.  Ejecuta las pruebas para verificar los flujos de autenticación y mensajes.
